# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行检测
- 提供完整的YOLOv8模型训练方案，包含3000张VOC2007数据集（7类目标：飞机、自行车、公交车、汽车、摩托车、人、火车）
- 提供训练好的模型及训练结果
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 支持自定义绘制检测边界线和检测区域，并记录目标数量
- 提供完整的UI界面，实时显示目标位置、目标总数和置信度信息

## 适用场景
- 行人车辆目标跟踪与计数
- 交通监控与分析
- 安防监控系统
- 智能停车场管理
- 城市交通流量统计

## 技术特点
- 基于YOLOv8深度学习框架
- 使用Pycharm/VSCode开发环境
- 主要依赖：Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9
- 提供完整的源码、UI界面文件和项目运行教程
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
